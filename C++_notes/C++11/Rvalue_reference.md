# 右值引用

- [ ] 解决不必要的复制动作，右边是一个右值引用的时候，左侧的值可以不必要直接做分配
赋值动作，可以直接从右边`steal`值。

- [ ] 不能被赋值的就是右值。

- [ ] 临时对象也是一种右值。

## move 之后实际上是指针的拷贝，原来的指针打断

- 应该应用在临时对象的赋值之上，例如

```C++
class A{

A(const A& str){...} /* 拷贝构造函数 */
A(A&& str){...}      /* move拷贝函数 */

};
/* 当然一般不会这么写，表达这个意思 */
A a = A();
```

- 用`std::move()`方法或是临时对象作为右值，会调用右值引用，指针拷贝。
